## Azure Security Center

Azure Security Center provides unified security management. It assesses your environment against security standards and provides recommendations.

**Key Features:**

- Secure Score: Overall security posture rating
- Recommendations: Prioritized security improvements
- Compliance: Built-in compliance assessments
## Azure Sentinel

Azure Sentinel is a cloud-native SIEM. It collects data from across your entire environment.

**Data Sources:** Azure AD, Office 365, Microsoft Defender, custom connectors

**Detection:** Built-in analytics rules, custom KQL queries, fusion engine

## Azure AD Security

Azure AD is the identity backbone. Key security features:

- Conditional Access: Context-aware access policies
- MFA: Multi-factor authentication enforcement
- PIM: Privileged Identity Management for just-in-time admin access
- Identity Protection: Risk-based sign-in policies

## Resource Group Security

Organize resources by security boundary:

- Separate production from development
- Apply different policies per resource group
- Use Azure Policy for compliance enforcement
- Monitor with Azure Monitor and Log Analytics
