## IOC-Based Hunting

Start with known IOCs: IP addresses from threat feeds, domain names from incidents, file hashes from malware analysis, email addresses from phishing campaigns.

## Threat Intel Feeds

Open source: OTX, MISP, Abuse.ch. Commercial: Recorded Future, CrowdStrike. Government: CISA alerts. Industry ISACs: sector-specific sharing.

## STIX/TAXII

STIX: Standard format for sharing threat intelligence. TAXII: Protocol for transporting STIX data.

## Intel-to-Detection Pipeline

1. Receive intelligence 
2. Validate and contextualize 
3. Create detection rules 
4. Deploy to SIEM/EDR 
5. Monitor for matches 
6. Refine based on results
