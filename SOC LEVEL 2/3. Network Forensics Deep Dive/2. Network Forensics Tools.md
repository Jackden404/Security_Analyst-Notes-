## NetworkMiner

NetworkMiner is an open-source Network Forensic Analysis Tool (NFAT). It extracts files, images, credentials, and session information from pcap files.

**Key Features:**

- Automatically extracts files transferred over the network
- Recovers deleted files from network traffic
- Identifies operating systems via TCP/IP fingerprinting
- Extracts credentials from HTTP, FTP, SMTP, Telnet

**Use Case:** You capture network traffic during an incident. NetworkMiner extracts 15 files that were transferred, including a Word document with a macro and an executable.

## Zeek (formerly Bro)

Zeek is a powerful network analysis framework that transforms raw traffic into structured logs. Unlike Wireshark, Zeek focuses on high-level analysis rather than packet-level inspection.

**Key Logs:**

- conn.log — All network connections
- dns.log — DNS queries and responses
- http.log — HTTP requests and responses
- ssl.log — TLS/SSL certificate information
- files.log — Files transferred over the network

**Use Case:** Zeek generates a conn.log showing 500 DNS queries to a suspicious domain within 10 minutes — a strong indicator of DNS tunneling.

## Arkime (formerly Moloch)

Arkime is a full packet capture and search system designed for large-scale deployments. It captures, indexes, and allows searching through massive amounts of network traffic.

**Key Features:**

- Full packet capture with indexed search
- Session tracking across connections
- Protocol decoding and extraction
- Integration with threat intelligence feeds

**Use Case:** During an investigation, you search Arkime for all traffic to a specific IP over the past 30 days and reconstruct the complete attack timeline.

## Choosing the Right Tool

|Tool|Best For|Scale|
|---|---|---|
|NetworkMiner|Quick file extraction from pcap|Small-medium|
|Zeek|Structured log generation|Medium-large|
|Arkime|Full packet capture and search|Large enterprise|
