## What EDR Actually Collects

When you deploy an EDR agent on an endpoint, it is not just watching for malware. It is recording everything happening on that machine — every process that starts, every file that gets written, every network connection that opens.

Think of it like a security camera for your computer, but instead of video, it captures structured data about system activity.

## The Four Pillars of EDR Telemetry

**Process Telemetry:** Every process launch is recorded — the executable name, command line arguments, parent process, and user who ran it. This is how you spot PowerShell running under a Word document, which is almost always malicious.

**File Telemetry:** Every file creation, modification, and deletion is tracked. When aransomware encrypts 500 files in 30 seconds, this telemetry shows you exactly which files were touched and in what order.

**Registry Telemetry:** Windows stores configuration in the registry. Malware often modifies registry keys for persistence — Run keys, Services, WMI subscriptions. EDR catches these changes.

**Network Telemetry:** Every outbound connection is logged — destination IP, port, protocol, bytes transferred. When malware calls home to a C2 server, EDR sees it.

## Reading EDR Data in Practice

Here is what a real EDR event looks like:

A Word document (WINWORD.EXE) spawns PowerShell with an encoded command. The PowerShell then downloads a file from an external URL and executes it.

In a traditional SIEM, you would see three separate events. In EDR, you see the entire process tree as one correlated event — parent, child, grandchild — all linked together. That is the power of EDR telemetry.

## EDR Agent Architecture

The agent sits on the endpoint and:

1. Collects telemetry locally (no network dependency for collection)
2. Buffers data if the network is down
3. Sends compressed data to the cloud/console
4. Can execute remote response commands when instructed

The agent is lightweight — typically 50-100MB RAM — and designed to have minimal impact on system performance.
