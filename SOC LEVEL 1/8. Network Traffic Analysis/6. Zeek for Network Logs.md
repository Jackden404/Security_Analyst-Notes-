While Wireshark acts as a microscopic protocol analyzer for individual connections, scaling packet capture across high-speed enterprise networks requires a different class of tool. **Zeek** acts as an agentless, structured network security monitoring engine.

---
## 🧭 What is Zeek?

Zeek is an open-source network security monitoring platform that translates raw packet streams into highly structured, tab-separated log files.

[!NOTE] **Historical Name:** Zeek was formerly known as: **Bro**. (Renamed to Zeek in 2018 to better align with the security community).

---
## 🗂️ Zeek Log Hives

Zeek does not record raw `.pcap` files. Instead, it parses packets in memory and writes categorized log databases, which are typically shipped to a SIEM.

|Zeek Log Name|Inspected Network Data|Forensic / Threat Hunting Value|
|---|---|---|
|**`conn.log`**|IP addresses, ports, protocol layers, duration, byte counts, and connection state.|🟢 **Core Log:** Used to map connection durations, locate port scans, and establish basic baselines.|
|**`dns.log`**|Domain name queries, lookup types, answer records, and query transaction IDs.|🟢 **C2 Detection:** Detects DNS tunneling, DGA domains, and dynamic C2 resolution.|
|**`ssl.log`**|TLS handshakes, Server Name Indications (SNI), certificate subjects, and JA3 hashes.|🟢 **SSL Telemetry:** Resolves target domains even when traffic is fully encrypted.|
|**`files.log`**|File hashes (MD5/SHA-256), MIME types, and source/destination connections.|🟢 **Payload Tracking:** Tracks malware downloads and unencrypted document transfers.|

[!IMPORTANT] **Fundamental Connection Logging:** The specific Zeek log file that tracks all network connections, including duration, bytes, and state, is: **`conn.log`**.

---
## ⚖️ Zeek vs. Wireshark: Microscope vs. Surveillance

Security operations centers use both Zeek and Wireshark in tandem, but they serve entirely different investigative roles:

```
[ Raw Packets / Mirrored Port ] ──► [ Zeek Parser ] ──► [ Structured Logs (conn.log, http.log) ] ──► [ SIEM Alerts ]
                                        │
                                        ▼ (If anomaly found)
                                [ Wireshark Deep Dive ] ──► [ Packet-Level Inspection ]
```

[!TIP] **Tool Comparison:** The primary operational difference between the two tools is: **Zeek generates structured logs for monitoring; Wireshark provides packet-level inspection**.

Zeek works like a surveillance camera, summarizing activity across thousands of hosts continuously. Wireshark acts as a microscope, dissecting the exact sequence of individual packet payloads to analyze exploits, session handshakes, or protocol details.
