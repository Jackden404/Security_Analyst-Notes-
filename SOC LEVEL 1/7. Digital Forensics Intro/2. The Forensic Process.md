### 1. Phase 1: Identification

[!IMPORTANT]В **Scoping the Scene:**В The first phase of the forensic process is:В **Identification**. Before touching any devices, the investigator must locate all potential sources of evidence (laptops, storage drives, cloud backups, RAM, mobile devices) and assess which data is the most volatile.

### 2. Phase 2: Preservation

This phase focuses on keeping the original evidence completely unaltered.

[!CAUTION]В **Write Blocking:**В AВ **write blocker**В (hardware or software) must be used during disk imaging:В **To prevent any modification to the original evidence**. Even simply plugging an evidence drive into a normal OS can write system files or update file access timestamps, violating evidence integrity. Hashing (SHA-256) is performed to verify that the forensic image matches the original perfectly.

### 3. Phase 3: Collection

Acquiring the physical and logical files. Volatile data (RAM dumps viaВ `WinPmem`В or network state tables) must be captured before turning off a system.

### 4. Phase 4: Examination

Using specialized forensic suites (likeВ `Autopsy`В orВ `EnCase`) to read file systems, extract hidden files, recover deleted files from unallocated space, and parse system metadata.

### 5. Phase 5: Analysis

Reconstructing the chronological timeline of events, correlating findings across multiple event logs, and drawing analytical conclusions about who, what, when, and how.

### 6. Phase 6: Reporting

Documenting findings in a clear, objective report suitable for both technical teams and legal stakeholders.

## рҹ“қ Documenting Accountability: Chain of Custody

Maintaining an audit trail is critical to demonstrating that the evidence was not tampered with, exposed, or modified while in custody.

[!TIP]В **Tracking Evidence:**В The primary administrative document that tracks who handled the evidence throughout the investigation is the:В **Chain of Custody form**. This form logs names, timestamps, physical locations, signatures, and hash values for every single handoff of the evidence.
