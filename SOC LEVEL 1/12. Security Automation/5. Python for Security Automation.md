## 🔌 API Integration: VirusTotal Example

Almost all modern security tools (SIEMs, firewalls, EDRs, active directories) expose **REST APIs**. Security engineers use Python libraries to bridge these tools together.

[!TIP] **HTTP Requests in Python:** The standard Python library used for making HTTP API calls is **requests**. The script below queries the VirusTotal v3 API to look up a file hash:

```python
import requests
import json

VT_API_KEY = "your_api_key_here"

def check_hash(file_hash):
    # Construct endpoint URL
    url = "https://www.virustotal.com/api/v3/files/" + file_hash
    headers = {"x-apikey": VT_API_KEY}
    
    # Send GET request using requests library
    response = requests.get(url, headers=headers)
    if response.status_code == 200:
        data = response.json()["data"]["attributes"]
        stats = data["last_analysis_stats"]
        detections = stats["malicious"]
        total = sum(stats.values())
        return {
            "hash": file_hash,
            "detections": detections,
            "total_engines": total,
            "detection_rate": f"{detections}/{total}"
        }
    else:
        return {"error": "Lookup failed", "status_code": response.status_code}

# Test with a dummy hash
result = check_hash("d55f983c994caa160ec63a59f6b4250fe67fb3e8c43a388aec60a4a6978e9f1e")
print(json.dumps(result, indent=2))
```

## 🔍 Log Parsing & Pattern Extraction

Log files come in various formats (JSON, syslog, CSV, or unstructured text). Security engineers often write scripts to scan logs for malicious activity.

[!WARNING] **Regular Expressions (Regex):** In a log parsing script, the purpose of regex patterns is **to match specific log structures and extract relevant fields** (like IPs, timestamps, request paths, and HTTP status codes). Using regex enables the parsing of unstructured strings into structured variables for programmatic analysis:


```python
import re
from collections import Counter

# Regex matching: IP - - [Date] "Request" Status Size
LOG_PATTERN = r"(\d+\.\d+\.\d+\.\d+) - - \[(.+?)\] \"(.+?)\" (\d{3}) (\d+)"

def parse_apache_log(log_file):
    ip_counter = Counter()
    suspicious = []
    
    with open(log_file, "r") as f:
        for line in f:
            match = re.match(LOG_PATTERN, line)
            if match:
                ip, timestamp, request, status, size = match.groups()
                ip_counter[ip] += 1
                
                # Check for SQL injection attempts in the request field
                if re.search(r"(union|select|insert|drop|delete|update).*from", request, re.I):
                    suspicious.append({"ip": ip, "type": "SQL injection attempt"})
                    
    return {"top_ips": ip_counter.most_common(10), "suspicious": suspicious}
```
