# Security Analyst Notes

A personal collection of notes, walkthroughs, and reference material compiled while learning and working as a SOC Analyst / Security Analyst. This repo is a living knowledge base covering foundational concepts, alert triage, investigation techniques, and digital forensics & incident response (DFIR).

## Repository Structure

| Folder | Description |
|---|---|
| [1.BackBone](./1.BackBone) | Core/foundational concepts every security analyst should know: networking, OS internals, logging, and other building blocks. |
| [2. Alert Investigation Specialist](./2.%20Alert%20Investigation%20Specialist) | Notes and playbooks on triaging and investigating security alerts. |
| [DFIR](./DFIR) | Digital Forensics & Incident Response notes: artifact analysis, evidence collection, and incident handling. |
| [SOC Basics](./SOC%20Basics) | Introductory SOC (Security Operations Center) concepts and terminology. |
| [SOC LEVEL 1](./SOC%20LEVEL%201) | Tier 1 analyst skills: initial alert monitoring, triage, and escalation. |
| [SOC LEVEL 2](./SOC%20LEVEL%202) | Tier 2 analyst skills: deeper investigation, threat hunting, and incident response support. |

## Purpose

This repository serves as:
- A personal study log while working through SOC/Security Analyst training and certifications
- A quick-reference guide for common investigation and DFIR workflows
- A resource others learning the same path may find useful

## How to Use

Browse the folders in a rough learning progression:
1.BackBone -> SOC Basics -> SOC LEVEL 1 -> 2. Alert Investigation Specialist -> SOC LEVEL 2 -> DFIR

Or jump directly to the topic you need.
